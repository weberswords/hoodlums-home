# Brand UI Kit — The Intelligent Hoodlums

High-fidelity brand/web UI kit for theintelligenthoodlums.com.

## Components

| File | Description |
|---|---|
| `Header.jsx` | Masthead band + nav bar with active state |
| `Hero.jsx` | Splash hero — Ink bg, badge, Director's Commentary |
| `PillarCards.jsx` | Design · Disrupt · Deliver pillar cards with hover states |
| `CaseStudy.jsx` | Tabbed case studies + House Rules grid |
| `Footer.jsx` | Credits box footer, Ink bg |
| `index.html` | Interactive multi-page prototype |

## Usage

Open `index.html` in a browser. Navigate between Home, Case Studies, and The Bond using the nav bar. State persists in localStorage on refresh.

## Design Width

1280px+. All components use fluid layouts with 64px horizontal padding.

## Notes

- No external icon library — badge asset from `assets/HM-badge-full.png`
- Fonts loaded from `fonts/` via `colors_and_type.css`
- All color values sourced from `colors_and_type.css` CSS vars
